ezSyntax

Easily add syntax highlighting to your <code> blocks using one line of code.

Getting Started (1)

Reference the JavaScript code between your HTML <head> and </head> tags: <script src="https://ezsyntax.willm.xyz/user/script.js"></script>

Syntax Highlight (2)

To highlight code when the page loads, add this to your <body> tag: <body onload="syntax('className', number)">

If you want to highlight code on click, add this instead: <body onclick="syntax('className', number)">

The number references to which class number you're wanting to highlight.
For example, if you wanted to highlight the first element with your selected class, you would do syntax('demo', 0)
